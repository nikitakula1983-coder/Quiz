Sci-Fi Chemistry Quiz (v2.0)
Интерактивный футуристичный тест по химии, посвященный анализу энергетических и структурных изменений материи.
Особенности
 * Стиль: Киберпанк / Научная лаборатория.
 * Контент: 20 вопросов по темам тепловых эффектов и типов химических реакций.
 * Система подсказок: Ссылки на страницы учебника (110–122).
 * Технологии: HTML5, Tailwind CSS, Google Fonts.
Как запустить
Просто откройте файл index.html в любом современном браузере.
Оценка
Результат рассчитывается автоматически по 10-балльной шкале.
